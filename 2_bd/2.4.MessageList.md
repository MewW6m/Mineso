# メッセージ一覧

## 項目定義

| 項目           | 型     | 桁数      | パターン                                     |
| -------------- | ------ | --------- | -------------------------------------------- |
| ID             | 文字列 | 9         | 種類1-種類2-連番                             |
| 種類1          | 文字   | 1         | V ... 画面<br />L ... ログ<br />E ... エラー |
| 種類2          | 数字   | 2 (0埋め) | -                                            |
| 連番           | 数字   | 4 (0埋め) | 連番                                         |
| メッセージ内容 | 文字列 | -         | -                                            |
